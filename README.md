## Instructions

### Stow dotfiles

1. `mv` each file or directory
Example: `mv ~/.zshrc ~/workspace/dotfiles/nickel/.zshrc`

### Apply dotfiles

1. `cd` into a subdirectory
2. `stow -t ~ nickel` to apply the entire directory
